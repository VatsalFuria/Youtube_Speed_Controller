A browser extension for optimizing youtube playback rates. 

Simplifies the process for me, with max playback rate upto 6x. 

Keyboard shortcuts for direct manipulation. (CTRL + SHIFT + Y -> toggle extension popup, SHIFT + </> arrow keys for 0.5 decrease/increase)

Easy interface, with convenient presets. (1x, 1.5x, 2.5x, 3.5x; 0.5x increase and decrease buttons; Slider with 0.1x step)

Persistant changes across all youtube videos and tabs.
